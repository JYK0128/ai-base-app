# 02. 애플리케이션 인증 서비스 (요약)

## 1. 개요

* 최종 소비자(End User)를 위한 고성능 소셜/간편 인증 및 온보딩 환경 제공

## 2. 주요 기능

* **간편 가입**: 주요 소셜 로그인(G/A/K/N) 및 매직 링크/OTP 지원
* **사용자 경험**: 가입 시 테넌트 자동 매핑 및 프로필/수신동의 관리
* **권한**: `End User` 기본 역할 및 파트너사별 서비스 레벨 접근 제어

## 3. 기술 요구사항

* **성능**: Redis 기반 스테이트리스 세션 및 대규모 동시접속 최적화
* **보안**: 개인정보 보호법 준수 및 OIDC/OAuth2 표준 프로토콜 적용

## 4. 핵심 API

* `signup/login`: 신규 가입 및 복합 인증 처리
* `profile`: 사용자 정보 관리 및 활동 이력 연동
* `withdraw`: 서비스 탈퇴 및 개인정보 파기 처리
