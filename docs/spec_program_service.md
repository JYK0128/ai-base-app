# Program Service 상세 사양서 (MVP Edition)

## 1. 개요

Program Service는 상품 카탈로그 및 각 조직의 구독 엔진을 담당합니다.

## 2. 비즈니스 로직

### 2.1. 구독 모델

* **요금제 관리**: 월간/연간 단위의 표준 구독 플랜을 운영합니다.
* **유예 기간**: 결제 실패 시 일정 기간 서비스를 유지하는 유예 기간 정책을 적용합니다.

## 3. 핵심 API 명세

| Method | Path | Description |
| :--- | :--- | :--- |
| `GET` | `/programs` | 전체 상품 카탈로그 조회 |
| `POST` | `/subscriptions` | 신규 구독 신청 |
| `PATCH` | `/subscriptions/:id/cancel` | 구독 해지 예약 |

## 4. 프로모션 엔진

* **할인 쿠폰**: 정액/정률 할인 쿠폰을 발행하여 마케팅에 활용할 수 있습니다.
* **이벤트 연동**: 특정 프로모션 시점에 구독료 할인을 적용하는 기능을 제공합니다.
