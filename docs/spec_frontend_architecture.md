# 프론트엔드 및 풀스택 아키텍처 사양서 (TanStack Start Edition)

## 1. 개요

최첨단 풀스택 프레임워크인 **TanStack Start**를 기반으로 서버 엔드포인트와 클라이언트 뷰의 경계가 없는 초고성능 타입 안전(Type-safe) 환경을 구축합니다.

## 2. 기술 스택 (The Modern Full-stack)

* **Framework**: **TanStack Start** (React-based Full-stack Framework)
* **Routing**: **TanStack Router** (Integrated Type-safe Routing)
* **Server Functions**: **Start Server Functions** (Type-safe RPC calling from Frontend)
* **Data Loaders**: **TanStack Start Loaders** (Server-side data fetching & Hydration)
* **State Management**: TanStack Query + Zustand
* **Form & Validation**: TanStack Form + Zod
* **Styling**: Tailwind CSS + Shadcn UI
