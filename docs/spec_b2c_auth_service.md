# B2C Auth Service 상세 사양서 (MVP Edition)

## 1. 개요

고객(Consumer)의 신원 식별, 다중 채널 로그인 및 통합 프로필 연동을 담당합니다.

## 2. 신원 관리 및 보안

* **SSO 통합 로그인**: Google, Apple, Kakao 등 주요 소셜 로그인을 통해 가입 허들을 낮춥니다.
* **JWT (JSON Web Token)**: 표준 JWT 기반 인증 및 갱신(Rotation) 로직을 적용하여 보안성을 확보합니다.
* **PII 암호화**: 이메일, 전화번호 등 개인정보는 DB 저장 시 양방향 암호화를 수행합니다.

## 3. 주요 기능

* **가입 및 탈퇴**: 약관 동의 및 이메일 기반 가입/탈퇴 프로세스.
* **프로필 관리**: 다중 프로그램(테넌트) 간의 통합 닉네임 및 설정 관리.
* **세션 관리**: Redis를 활용한 단일 세션 유지 및 강제 로그아웃 기능.

## 4. 핵심 API 명세

| Method | Path | Description |
| :--- | :--- | :--- |
| `POST` | `/auth/login` | 소셜 토큰 기반 인가 및 서비스 토큰 발급 |
| `POST` | `/auth/refresh` | 리프레시 토큰을 통한 액세스 토큰 갱신 |
| `GET` | `/profile` | 본인의 통합 프로필 정보 조회 |
| `DELETE` | `/profile` | 서비스 탈퇴 및 데이터 삭제 신청 |
