# 06. 플랫폼 어드민 포탈 스펙 (요약)

## 1. 개요

* 플랫폼 전반의 인프라, 파트너 계약 및 시스템 건전성을 총괄 관리하는 마스터 시스템

## 2. 주요 기능

* **파트너 라이프사이클**: 가입 신청 승인/반려, 테넌트별 리소스 할당량(Quota) 및 구독 상태 수동 제어
* **통합 관제 대시보드**: 전사 지표(매출, 트래픽) 시각화 및 마이크로서비스별 헬스체크/알림 수신
* **정책 운영**: 플랫폼 전체 이용약관 관리 및 전역 시스템 공지사항 발송

## 3. UI/UX 및 기술 표준

* **가시성**: 대량의 파트너 정보를 효율적으로 관리하기 위한 고성능 필터링 및 다크 모드 대시보드
* **프레임워크**: TanStack Start 기반의 고성능 SSR/SPA 환경 구축
