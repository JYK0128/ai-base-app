# 07. 파트너 어드민 포탈 스펙 (요약)

## 1. 개요

* 파트너사가 자사 운영진, 구독 서비스 및 고객(Client)을 독립적으로 관리하는 전용 대시보드

## 2. 주요 기능

* **조직 관리**: 자사 직원 초대 및 세부 직무 권한(Role) 부여, 직원 활동 감사 로그 제공
* **서비스 설정**: 구독 중인 멤버십 현황 확인, 서비스 기능 ON/OFF 및 화이트라벨링(로고, 색상, 도메인) 적용
* **Client 관리 (CRM)**: 최종 사용자 활동 데이터 조회, 1:1 고객 문의 응대 및 비즈니스 통계 리포트

## 3. UI/UX 및 기술 표준

* **사용성**: 비즈니스 지표 가독성을 높인 위젯 기반 대시보드 및 실시간 고객 알림 채널 지원
* **프레임워크**: TanStack Start 기반의 반응형 웹 환경 제공
